A. LilyPond grammar

This appendix contains a description of the LilyPond grammar, as output from the parser. 

Grammar

    1 start_symbol: lilypond

    3 start_symbol: "#{"  embedded_lilypond

    4 lilypond: /* empty */

    6         | lilypond assignment
