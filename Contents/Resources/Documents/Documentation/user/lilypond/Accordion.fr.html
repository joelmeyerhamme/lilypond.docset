Symboles de jeux

De nombreux accordéons possèdent plusieurs jeux d’anches capables de jouer à l’unisson ou bien à l’octave inférieure ou supérieure par rapport aux notes écrites. Chaque facteur d’accordéon donne des noms différents aux tirettes (shifts) qui permettent de sélectionner les différents jeux d’anches tels que hautbois, musette ou bandonéon, de telle sorte qu’un système de symbole a fini par voir le jour afin de faciliter les instructions pour exécuter un morceau.

Accordion-discant symbols

Accordion discant-specific symbols are added using \markup. The vertical placement of the symbols can be tweaked by changing the \raise arguments. 

discantus = \markup {
  \musicglyph #"accordion.accDiscant"
}
point = \markup {
  \musicglyph #"accordion.accDot"
}

\layout { ragged-right = ##t }

% registre de 16 pieds
accBasson = ^\markup {
  \combine
  \discantus
  \raise #0.5 \point
}

% een korig 8 en 16 voets register
accBandon = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \raise #1.5 \point
}

accVCelle = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \combine
        \raise #1.5 \point
        \translate #'(1 . 0) \raise #1.5 \point
}

% ergistre 4-8-16 pieds
accHarmonica = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \combine
        \raise #1.5 \point
        \raise #2.5 \point
}

accTrombonne = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \combine
        \raise #1.5 \point
        \combine
          \translate #'(1 . 0) \raise #1.5 \point
          \translate #'(-1 . 0) \raise #1.5 \point
}

% eenkorig 4 en 16 voets register
accOrgue = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \raise #2.5 \point
}

accMaitre = ^\markup {
  \combine
    \discantus
    \combine
      \raise #0.5 \point
      \combine
        \raise #1.5 \point
        \combine
          \translate #'(1 . 0) \raise #1.5 \point
          \combine
            \translate #'(-1 . 0) \raise #1.5 \point
            \raise #2.5 \point
}

accAccordeon = ^\markup {
  \combine
    \discantus
    \combine
      \raise #1.5 \point
      \combine
        \translate #'(1 . 0) \raise #1.5 \point
        \combine
          \translate #'(-1 . 0) \raise #1.5 \point
          \raise #2.5 \point
}

accMusette = ^\markup {
  \combine
    \discantus
    \combine
      \raise #1.5 \point
      \combine
        \translate #'(1 . 0) \raise #1.5 \point
        \translate #'(-1 . 0) \raise #1.5 \point
}

accCeleste = ^\markup {
  \combine
    \discantus
    \combine
      \raise #1.5 \point
      \translate #'(-1 . 0) \raise #1.5 \point
}

accHautbois = ^\markup {
  \combine
    \discantus
    \combine
      \raise #1.5 \point
      \raise #2.5 \point
}

accClarinette = ^\markup {
  \combine
    \discantus
    \raise #1.5 \point
}

accPiccolo = ^\markup {
    \combine
       \discantus 
       \raise #2.5 \point
}

accViolon = ^\markup {
  \combine
    \discantus
    \combine
      \raise #1.5 \point
      \combine
        \translate #'(1 . 0) \raise #1.5 \point
        \raise #2.5 \point
}

\relative c'' {
  c4 d\accBasson e f
  c4 d\accBandon e f
  c4 d\accVCelle e f
  c4 d\accHarmonica e f
  c4 d\accTrombonne e f
  \break
  c4 d\accOrgue e f
  c4 d\accMaitre e f
  c4 d\accAccordeon e f
  c4 d\accMusette e f
  c4 d\accCeleste e f
  \break
  c4 d\accHautbois e f
  c4 d\accClarinette e f
  c4 d\accPiccolo e f
  c4 d\accViolon e f
}
